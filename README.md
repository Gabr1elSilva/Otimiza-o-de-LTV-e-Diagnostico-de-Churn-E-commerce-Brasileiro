# Otimiza-o-de-LTV-e-Diagnostico-de-Churn-E-commerce-Brasileiro
Análise de 100k pedidos da Olist diagnosticou a 'Armadilha Transacional': 97% dos clientes não retornam. Usando Python, provei que atrasos logísticos destroem a fidelidade mais que o preço. O projeto define estatisticamente o 'Churn Silencioso' e propõe ações via RFM para estancar a perda de receita.
